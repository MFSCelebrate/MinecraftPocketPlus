# 介绍
该仓库所存储的是 **Minecraft Pocket Edition 0.6.1** 的源码修改版，支持Armv8和v7与Windows，整个改版的宗旨就是以探索边境之地与距离现象为目的，解开携带版非无限世界的边境之地谜团。

# 为什么要这么做？
由于Minecraft的源码泄露，这无疑是给了我们修改携带版的一个机会，以此来模拟基岩版生成器是最好过的了。
其实作者是服务器插件开发者转生来的
